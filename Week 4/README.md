# Week 4 — LSTM Language Model

Character-level language model built using a 2-layer LSTM on the Tiny Shakespeare dataset.

This notebook follows the same language modeling pipeline as Karpathy's GPT lecture, replacing the Transformer with an LSTM for sequential modeling.

### Features

* Character-level tokenizer
* Tiny Shakespeare dataset
* PyTorch `nn.LSTM`
* Autoregressive text generation
* Gradient clipping
* GPU support (if available)

### Run

Open `lstm_language_model.ipynb` in Google Colab or Jupyter Notebook and run the cells sequentially.
